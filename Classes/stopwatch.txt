## 📝 오늘의 파이썬 클래스 문제: ⏱️ Stopwatch 클래스 만들기
시간의 경과를 측정하는 스톱워치 클래스를 만들어 보세요.

요구사항:
__init__(self):

객체가 처음 만들어질 때 필요한 인스턴스 변수들을 초기화합니다.

start_time: 시작 시간을 저장할 변수, 초기값은 None.

end_time: 종료 시간을 저장할 변수, 초기값은 None.

running: 스톱워치가 현재 실행 중인지 상태를 저장할 변수(boolean), 초기값은 False.

start(self):

만약 스톱워치가 이미 실행 중(running이 True)이면, "스톱워치가 이미 실행 중입니다."라고 출력합니다.

그렇지 않다면, running을 True로 바꾸고 start_time에 현재 시간을 기록한 뒤 "스톱워치가 시작되었습니다."라고 출력합니다.

힌트: import time 후 time.time()을 사용하세요.

stop(self):

만약 스톱워치가 실행 중이 아니면(running이 False), "스톱워치가 시작되지 않았습니다."라고 출력합니다.

실행 중이라면, running을 False로 바꾸고 end_time에 현재 시간을 기록한 뒤 "스톱워치가 멈췄습니다."라고 출력합니다.

reset(self):

모든 인스턴스 변수(start_time, end_time, running)를 맨 처음의 초기값으로 되돌립니다.

"스톱워치가 초기화되었습니다."라고 출력합니다.

get_elapsed_time(self):

스톱워드로 측정된 경과 시간을 출력합니다.

실행 중일 때: 현재 시간과 start_time의 차이를 계산해서 f"경과 시간: O.XX초" 형식으로 출력합니다.

멈춘 상태일 때: end_time과 start_time의 차이를 계산해서 f"최종 경과 시간: O.XX초" 형식으로 출력합니다.

시작한 적이 없을 때: "측정 기록이 없습니다."라고 출력합니다.

힌트: 소수점 둘째 자리까지 표현하려면 f"{계산결과:.2f}" 형식을 사용하세요.
